# Wiki Schema

| type | Purpose | Default directory |
| --- | --- | --- |
| `source` | Summary and analysis of one source | `wiki/sources/` |
| `concept` | Reusable concept, model, or term | `wiki/concepts/` |
| `domain` | Topic map and navigation page | `wiki/domains/` |
| `project` | Project, learning plan, or experiment | `wiki/projects/` |
| `question` | Open or tracked question | `wiki/questions/` |
| `synthesis` | Cross-source synthesis or current conclusion | `wiki/syntheses/` |
| `daily` | Daily note | `daily/` |
| `index` | Navigation or system entry | root or `ops/` |
| `log` | Change log | `wiki/log.md` |

| status | Meaning |
| --- | --- |
| `seed` | Newly created with only basic structure |
| `growing` | Accumulating material and links |
| `stable` | Stable enough to use as a reference entry |
| `archive` | Kept but no longer actively maintained |
