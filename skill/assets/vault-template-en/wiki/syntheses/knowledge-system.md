# Knowledge Iteration System

This system separates knowledge work into three layers:

1. `raw/` stores original input.
2. `wiki/` stores understood and linked knowledge.
3. `ops/` and `AGENTS.md` store maintenance rules so humans and AI can iterate the same system over time.

## Iteration Loop

- Capture: save first; do not rush to perfect organization.
- Ingest: turn one source into a referenceable source page.
- Link: extract concepts, questions, and projects, then connect them to existing pages.
- Synthesize: regularly combine multiple sources into higher-level judgments.
- Lint: check isolated pages, missing sources, duplicate concepts, and stale questions.
