# AGENTS.md

本库是一个个人 wiki 知识迭代系统，用来管理日常知识积累。默认用中文写作，除非用户明确要求英文。

## 核心分层

- `raw/`: 原始材料层。放网页摘录、PDF、截图、临时记录、转录文本等，尽量追加，不改写事实来源。
- `wiki/`: 知识层。放经过消化、链接、重写后的笔记，是日常查询和复盘的主要入口。
- `templates/`: Obsidian 模板层。新建笔记时优先复用这里的结构。
- `ops/`: 操作层。放工作流、schema、提示词和维护规则。

## 工作原则

- 先读 `首页.md`、`wiki/index.md`、`ops/workflows.md`，再编辑知识库。
- 保持 `raw -> wiki -> index/log` 的迭代闭环。
- 不把未经支持的摘录升级为事实。重要判断要写明依据，或标记为 `待验证`。
- 新 wiki 页面必须包含 YAML frontmatter，并尽量使用 `[[双链]]` 连接相关页面。
- 页面标题短而稳定；文件名优先使用小写英文和连字符，中文标题放在 frontmatter 的 `title` 字段中。
- 每次完成较明显的整理、合并、重命名或新增索引，都更新 `wiki/log.md`。
