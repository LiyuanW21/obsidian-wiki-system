# Wiki Schema

| type | 用途 | 默认目录 |
| --- | --- | --- |
| `source` | 单个来源的摘要和分析 | `wiki/sources/` |
| `concept` | 可复用概念、模型、术语 | `wiki/concepts/` |
| `domain` | 主题地图和导航页 | `wiki/domains/` |
| `project` | 项目、学习计划、实验 | `wiki/projects/` |
| `question` | 开放问题和追踪问题 | `wiki/questions/` |
| `synthesis` | 跨来源综合、阶段性判断 | `wiki/syntheses/` |
| `daily` | 每日记录 | `daily/` |
| `index` | 导航和系统入口 | 根目录或 `ops/` |
| `log` | 变化记录 | `wiki/log.md` |

| status | 含义 |
| --- | --- |
| `seed` | 刚创建，只有基本结构 |
| `growing` | 正在积累材料和链接 |
| `stable` | 结构稳定，可以作为引用入口 |
| `archive` | 保留但不再主动维护 |
